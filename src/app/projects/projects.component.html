Smokedetector using JAVA
A simple JAVA based device which can be used by the traffic constables to rectify a person who is driving is Drunk or not. It will detect through the smell of the mouth. It will indicate by giving the Number Readings. If the number reading exceeds 60, then the person is Drunk, otherwise fine.
